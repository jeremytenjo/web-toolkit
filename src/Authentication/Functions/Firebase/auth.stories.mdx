# Continue with Social account

Log in with a social media account.

Providers: Google, Facebook

Defaults: `provider = 'google'`

Returns `user` or `error`

### Usage

```js
import authSocial from @tenjojeremy/web-toolkit/build/Authentication/Functions/Firebase/auth.social
```

```js
res = await authSocial({ provider: 'google' })
```

# Continue with Email

Login or Register with a email.

Defaults: `action = 'login'` | `email = ''` | `register = ''`

Returns `user` or `error`

### Usage

```js
import authEmail from @tenjojeremy/web-toolkit/build/Authentication/Functions/Firebase/auth.email
```

```js
const { user, error } = await authEmail({
  action: 'login',
  email: 'test@gmail',
  password: 'password1246',
})
```
