# Continue with Social account

Log in with a social media account.

Providers: Google, Facebook

### Usage

```js
import authSocial from @tenjojeremy/web-toolkit/build/Authentication/Functions/Firebase/auth.social
```

```js
res = await authSocial({ provider: 'google' })
```
